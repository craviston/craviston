# **App Name**: Craviston

## Core Features:

- Punchy Hero Experience: High-impact 3-line headline with bold CTA directing users to Zomato for delivery.
- Dynamic Menu Filter: A responsive lineup featuring waffles and sandwiches with tab-based filtering to instantly browse categories.
- CraveMatch AI Tool: An AI-powered flavor matcher tool that suggests the perfect waffle or sandwich combination based on the user's current mood or flavor preferences.
- Scrolling Hype Ticker: A marquee animation strip displaying brand taglines and the core delivery promise to maintain street-food energy.
- Trust & Velocity Stats: Dynamic grid highlighting delivery speeds and top ratings to build immediate consumer confidence.
- Zomato-Locked Conversion Section: A high-conversion footer section styled in Zomato’s signature red (#E23744) to maximize click-throughs.

## Style Guidelines:

- Primary color: A bold Burnt Orange (#D44D25) that evokes warmth and spicy street energy.
- Background color: Warm Peach Cream (#F8E9D4) used as the dominant theme as requested by the user.
- Accent: Deep Mocha (#432E27) used for heavy high-contrast sections and large typography.
- Headline: 'Bebas Neue' or 'Anton' for a chunky, oversized, bold display feel. Body: 'Nunito' or 'DM Sans' for clean, readable secondary text. Note: currently only Google Fonts are supported.
- Graphic food emojis and oversized watermark text (e.g., 'CRAVISTON') used as visual motifs rather than generic stock imagery.
- Maximalist, full-bleed alternating sections with large white-space and bold typographic heirarchy inspired by modern creative agencies.
- Smooth scroll-reveal animations using IntersectionObserver to slide elements into view as the user scrolls.
